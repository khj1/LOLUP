# 롤업

롤업은 인기 게임 리그오브레전드 유저 매칭 서비스입니다.
유저들은 원하는 파트너를 찾기위해 다양한 커뮤니티에 모집글을 올리곤 했습니다.
유저 매칭 플랫폼을 형성하여 유저들의 불편함을 해소하기 위해 '롤업'을 기획하게 되었습니다.
